*vim-symbolize* Convert between Ruby symbols and strings

INTRODUCTION                  *vim-symbolize*

This plugin provides two key bindings, one to convert
a Ruby symbol like :abc into a string like "abc", and
another to convert a string back to a symbol.


USAGE

  nmap <leader>sa <Plug>SymToStr
  nmap <leader>ss <Plug>StrToSym
